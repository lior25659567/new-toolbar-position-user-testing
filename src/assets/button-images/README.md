# Button Images Organization

This folder contains all images organized by their associated toolbar button functionality.

## Folder Structure

### `/default/`
Images shown in the main center view when entering the scan page (no buttons active):
- `dental-arch-color.png` - Main color view of dental arch
- `dental-arch-grayscale-feedback.png` - Grayscale with blue feedback markers
- `dental-arch-color-feedback.png` - Color with feedback markers
- `3d-model-upper.png` - Upper 3D dental model
- `3d-model-lower.png` - Lower 3D dental model

### `/review-tool/` (Button 1)
Images for Review Tool button (NIRI/IOC comparison):
- `upper-arch-color.png` - Upper arch color image
- `lower-arch-niri.png` - Lower arch NIRI scan
- `occlusal-view-old.png` - Occlusal (top-down) view

### `/occlusalgram/` (Button 2)
Images for Occlusalgram button:
- (Currently empty - uses scale component)

### `/margin-line/` (Button 3)
Images for Margin Line button:
- `dental-arch-margin-view.png` - Center view when margin line is active

### `/prep-qc/` (Button 4)
Images for Prep QC button:
- `prep-model-1.png` - Prep quality check model 1
- `prep-model-2.png` - Prep quality check model 2

### `/monochrome/` (Button 0)
Images for Monochrome toggle:
- `dental-arch-grayscale.png` - Grayscale version of dental arch

### `/trim/` (Button 5)
Images for Trim functionality:
- `trim-icon.png` - Trim area visualization
